# Daily Next

Open or create the next weekday's daily note in [Obsidian](https://obsidian.md) — one command, no prompts, skips weekends automatically.

## Features

Reads your Core Daily Notes folder, date format, and template. No extra configuration needed.

- **Weekend-aware** — Friday goes to Monday; Saturday and Sunday go to Monday
- **Creates if missing** — applies your configured template on first open
- **Zero config** — picks up Core Daily Notes settings automatically
- **Mobile-ready** — no Electron or Node APIs required

## Install

**Manual:**

1. Download `main.js`, `styles.css`, and `manifest.json` from the [latest release](https://github.com/visualhue/obsidian-daily-next/releases/latest)
2. Copy the files to `<vault>/.obsidian/plugins/daily-next/`
3. Enable **Daily Next** in **Settings → Community plugins**

**BRAT:**

```
visualhue/obsidian-daily-next
```

Install [BRAT](https://github.com/TfTHacker/obsidian42-brat), add the repo above, then enable **Daily Next** in **Settings → Community plugins**.
